# Definitions: 
#   1. Entity: A person/location/organization/country...
#   2. Class: A set of entity, usually represented by hierarchy
#   3. Instance: a instance of a entity
#   4. Relationship: a function among entity/class, i.e. 
#       C(e1, c1) --> e1 belong to c1
#       S(c1, c2) --> c1 is subclass of c2
#   
#
# Text Format: <entity>\t<instance>|<instance>\t<class1>|<class2>
# Example: 
#   苹果    苹果|Apple  品牌
#   想买    想买|想败|求推荐 商品特征#购买
#   辣      辣|火辣辣  商品特征#口味
#   中奖    中奖|中大奖 商品特征#促销
#   赠品    赠品|赠品   商品特征#促销
#   好喝    好喝        商品特征#口味
#   大赞    赞啊|大赞   情感词#正面
#
# 现有主要Class定义: 
#   人物#<人物类别细分>(Person)
#   地点#<地点类别细分>(Location)
#   机构#<机构类别细分>(Organization)
#   品牌#<品牌类别细分>(Brand)
#   商品型号(P-Model)
#   商品类型(P-Type)
#   商品特征#<特征类别细分>(P-Feature)
#   商家类型(S-Type)
#   商家特征#<特征类别细分>(S-Feature)
#   情感词#<情感词类别细分>(Sentiment)
#   食品#<食品类别细分>(Food)
